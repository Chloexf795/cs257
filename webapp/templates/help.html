Crime Data API - Help

Welcome to the Crime Data API! Below are the available endpoints and their usage:

1. GET /
    - Description: Returns a simple welcome message.
   
1. GET /help
    - Description: This page

2. GET /areas
    - Description: Retrieves a list of all available crime areas.

3. GET /types
    - Description: Retrieves a list of all crime types.

4. GET /dates
    - Description: Retrieves a list of all recorded crime dates.

5. GET /rawcsv
    - Description: Downloads the entire crime dataset in CSV format.

6. GET /crimes
    - Description: Retrieves crime data filtered by date, area, and/or crime type.
    - Query Parameters:
        - start_date: Filter crimes that occurred on or after this date (yyyy-mm).
        - end_date: Filter crimes that occurred on or before this date (yyyy-mm).
        - locations: Filter crimes from a specific location.
        - type: Filter crimes by the crime type.
    - Example: curl "/crimes?start_date=2025-01&end_date=2025-02&area=Topanga&type=BATTERY%20-%20SIMPLE%20ASSAULT"
